# ExoTrooper (aaExamenFinal)

**Autor:** Anthonny Almeida  
**Repositorio:** https://github.com/Anthonio16/aaExamenFinal.git

## Descripción
Aplicación **ExoTrooper** desarrollada en **Java** siguiendo arquitectura **N-Tier (N-Layer)**.  
Permite autenticación por consola (máx. 3 intentos) y, tras acceso correcto, muestra una interfaz gráfica tipo prototipo para gestionar Exobots:
- Agregar TipoExobot a la grilla
- Buscar por TipoExobot
- Entrenar arma (alternando entrenamiento con soldado experto)
- Ejecutar acción del arma y registrar eventos
